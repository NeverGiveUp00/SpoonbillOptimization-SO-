#MATLAB Scripts

This repository contains a collection of MATLAB scripts for a meta-heuristic algorithm.
## 🚀 How to Run

It is extremely simple. **Just run the `main.m` file.**

1. Download this repository (Click the green `Code` button -> `Download ZIP`) and extract it.
2. Open MATLAB and navigate to the extracted folder in the "Current Folder" panel.
3. In the Command Window, type `main` and press Enter. 
   *(Alternatively, right-click `main.m` in the file browser and select "Run")*.

## 📂 File Structure

- `main.m`: The main entry point. **Run this file.**
- `*.m` (other 5 files): Helper functions and sub-routines automatically called by the main script.

## 💻 Requirements

- MATLAB (R2020a or later recommended).
- No specific toolboxes are strictly required.
